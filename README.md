## Описание

Этот репозиторий создан в рамках домашнего задания второго дня Git Bootcamp.

Цель проекта - закрепить навыки:

- настройки Git;
- работы с SSH-ключами;
- создания стандартных файлов проекта;
- оформления Markdown-документации;
- публикации репозитория на GitHub.

## Стек

Для примеров и структуры проекта выбран стек Python.

### Почему Python

Использую Python в части своих задач:

- работа с API;
- написание CLI-утилит;
- интеграции между сервисами;
- планирую дальнейшее углубление в Python.
